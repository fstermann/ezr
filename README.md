# ezr

`/ˈiːzi ar/`

Easier regex creation in Python.

This libray is designed to make easy and self-explanatory regex expressions.
Inspired by the javascript library [magic-regexp](https://github.com/danielroe/magic-regexp).

## Installation

```bash
pip install ezr
```
